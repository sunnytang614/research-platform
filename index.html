【基础调研】视频号竞品深度分析提示词

用于生成VP级竞品分析报告，专门针对视频号明星业务 

你是一名顶级互联网公司战略分析负责人 + McKinsey咨询顾问。
请针对【视频号明星业务】的竞品（抖音、小红书、Instagram），输出一份"战略级竞品分析报告"。

【一、执行摘要】
- 用3-5条核心结论总结本报告
- 每条必须是"判断 + 依据 + 含义"

【二、竞品战略定位分析】
1. 抖音明星业务的战略定位：为电商服务的流量引擎
2. 小红书明星业务的战略定位：为品牌转化服务的种草平台
3. Instagram明星业务的战略定位：视觉影响力交易市场
4. 视频号明星业务应该如何定位？

【三、竞品商业模式分析】
1. 收入结构对比（广告、电商、订阅、分成）
2. 产品化能力对比（SKU标准化程度）
3. ROI与数据能力对比
4. 平台与创作者分成机制对比

【四、核心能力对比】
1. 供给控制能力对比
2. 分发能力对比（算法推荐、社交裂变、搜索）
3. 数据能力对比
4. 商业基础设施对比
5. 用户心智对比

【五、差异化定位建议】
1. 视频号的核心优势是什么？
2. 视频号应该如何与竞品差异化？
3. 短期、中期、长期战略建议是什么？

输出要求：
- 使用表格进行对比分析
- 每个观点必须有数据支撑
- 结论先行，不要废话
- 适合直接用于高管汇报

【基础调研】明星深度访谈提纲提示词

用于生成明星深度访谈提纲，了解艺人对线上价值的认知、合作痛点、改进建议 

请为【{艺人名称}】生成一份深度访谈提纲，访谈时长60分钟。

访谈目标：
1. 了解艺人对线上价值的认知
2. 了解艺人与视频号合作的痛点
3. 了解艺人对改进的建议

访谈问题框架：
【第一部分：背景与认知】(15分钟)
1. 请简单介绍一下您的演艺生涯和主要成就
2. 您对线上娱乐的初步印象是什么？
3. 您认为线上娱乐对艺人的价值是什么？
4. 您之前有没有在线上举办过活动？体验如何？

【第二部分：合作体验】(20分钟)
5. 您与视频号合作的体验如何？
6. 合作过程中遇到的最大挑战是什么？
7. 您认为合作模式有什么可以改进的地方？
8. 您对合作收益满意吗？为什么？

【第三部分：价值认知】(15分钟)
9. 您认为线上活动与线下活动的价值对比如何？
10. 您对线上收入的期望是多少？
11. 您愿意与平台建立长期合作吗？为什么？
12. 您认为什么样的合作模式是理想的？

【第四部分：改进建议】(10分钟)
13. 您对平台有什么改进建议？
14. 您希望平台提供什么支持？
15. 您对未来的合作有什么期待？

访谈注意事项：
- 创造轻松的访谈氛围
- 深入挖掘背后的原因
- 鼓励艺人分享真实想法
- 做好访谈记录

【基础调研】视频号数据指标体系提示词

用于生成完整的视频号明星业务数据指标体系 

请为【视频号明星业务】生成完整的业务数据指标体系。

【流量指标】
- DAU（日活跃用户数）
- MAU（月活跃用户数）
- 日均使用时长
- 留存率（次日/7日/30日）

【内容指标】
- 明星数量
- 合作次数
- 内容数量
- 内容质量评分

【商业化指标】
- GMV（交易总额）
- 客单价
- 转化率
- 客户数量
- 复购率

【社交裂变指标】
- 分享率
- 裂变系数
- 朋友圈曝光量
- 话题讨论量

【用户满意度指标】
- NPS（净推荐值）
- 用户满意度
- 用户投诉率
- 用户流失率

请为每个指标说明：
1. 指标定义
2. 计算方法
3. 目标值（3个月、6个月、12个月）
4. 监控频率（日/周/月）
5. 责任人
